## 🆘 遇到问题?

### 常见问题和解决方案

#### 问题: "Maven not found"

```
解决:
1. 重启 PowerShell/Terminal
2. 运行诊断工具
3. 查看 ENVIRONMENT_SETUP.md
```

#### 问题: "PostgreSQL connection failed"

```
解决:
1. 检查 PostgreSQL 是否运行: pg_isready
2. 查看数据库初始化步骤
3. 验证数据库密码
```

#### 问题: "Port already in use"

```
解决:
Windows: netstat -ano | findstr :5173
Linux:   lsof -i :5173
```

更多: 查看 [QUICK_START.md - 常见问题解决](QUICK_START.md#-常见问题解决)
